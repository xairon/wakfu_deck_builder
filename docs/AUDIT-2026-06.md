- **Conflit d'édition multi-appareils (last-write-wins)** : mitigé (gardes réseau/catalogue + flush avant logout réduisent la fenêtre). Une résolution « propre » (horodatage par entrée / fusion) est une vraie feature à concevoir — un merge hâtif risquerait de **créer** des bugs (cartes supprimées qui reviennent). À faire à part si besoin.
- **Image « Otomaï l'Enchanteur »** : aucune source récupérable (lien wtcg cassé) → repli gracieux sur le dos de carte (1 carte / 1585).
- **Vraies miniatures (thumbs)** : la perf de rendu est traitée (content-visibility + lazy) ; générer des vignettes ~256px reste une optim de bande passante à part.
- **`sync.ts` (multi-onglets)** : code mort mais testé et inoffensif — conservé (la cohérence est déjà assurée par Supabase).

**Note PWA** : le service worker (`registerType: autoUpdate`) sert l'ancien bundle jusqu'au rechargement suivant — les utilisateurs existants reçoivent les correctifs automatiquement à la prochaine visite.
